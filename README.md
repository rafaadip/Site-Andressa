# Site — Dra. Andressa Correia

Landing page profissional da **Dra. Andressa Chaves Correia** — médica com atuação
em **clínica geral** e **nutrologia**, com agendamento de **telemedicina** pelo WhatsApp.

## ✨ Recursos

- Página única, responsiva (mobile-first) e acessível
- Botão de WhatsApp no header, no hero, em CTAs e botão flutuante
- Seções: Sobre, Áreas de atuação, Como funciona a telemedicina, Formação e Contato
- Animações suaves (com suporte a `prefers-reduced-motion`)
- Sem dependências de build — HTML, CSS e JS puros

## 📁 Estrutura

```
index.html          → estrutura da página
css/styles.css      → estilos (design system sage/teal + creme)
js/main.js          → menu mobile, header e animações
assets/             → imagens (adicione a foto aqui)
```

## 🖼️ Adicionar a foto da Dra. Andressa

Coloque a foto em `assets/dra-andressa.jpg` (quadrada, ~800×800px).
Veja `assets/README.md`. Sem a foto, o site mostra um fallback com as iniciais.

## 🚀 Como visualizar

Basta abrir o `index.html` no navegador. Para um servidor local:

```bash
python3 -m http.server 8000
# acesse http://localhost:8000
```
