# Imagens do site

## Foto da Dra. Andressa

Coloque a foto de perfil aqui com o nome exato:

```
assets/dra-andressa.jpg
```

- Formato recomendado: **JPG** ou **WebP**
- Proporção ideal: **quadrada (1:1)** — ex.: 800×800px
- A mesma foto enviada no WhatsApp/LinkedIn funciona perfeitamente.

Enquanto a foto não for adicionada, o site mostra automaticamente um
quadro com as iniciais "AC" (fallback elegante), sem quebrar o layout.
